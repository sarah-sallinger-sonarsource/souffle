Detailed Description

Make indexable operations to indexed operations.

The transformer assumes that the RAM has been levelled before. The conditions that could be used for an index must be located immediately after the scan or aggregrate operation.

QUERY
...
FOR t1 in A
IF t1.x = 10 /\ t1.y = 20 /\ C
...

will be rewritten to

QUERY
...
SEARCH t1 in A INDEX t1.x=10 and t1.y = 20
IF C
...
